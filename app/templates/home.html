Sparse, but Functional

4/24/2014 | 2PM EST

Obviously design is not a strength of mine at the moment. This thing is very Geocities, but it works. Just learning how to get a website together with Python and Flask and getting it to actually be on the internet somewhere has been very educational. I'm using what I'm learning to cobble together a suite of extremely simple web apps to (hopefully) make a few things at work a bit easier or faster. I have two in mind right now: a template using HTML forms and Python to draft a boilerplate Outlook email, and a program to quickly tell if an institution's DEA license has expired. I'm sure I can think of more as I get deeper into Flask. I should probably also learn some jQuery/get back to JavaScript and figure out how to integrate that with Flask at some point to spice this gross little website up. Always something more to learn.

Blogging is not really my style but it's extremely helpful as far as practicing and learning more programming concepts/technologies. Sometimes I just feel like Creed Bratton.
