ScreenBloom

screenbloom.com
ScreenBloom Repository
Technologies Used:

Python, Flask, HTML, CSS, Javascript, jQuery

ScreenBloom parses an image of your current PC screen and extracts the average color of each pixel. That color is then sent to your connected Philips Hue bulbs to create a subtle Ambilight-like effect.

I had always been curious to replicate Philips' Ambilight effect available on many of their TVs, but was not able to find a suitable third-party solution, so I set about developing my own. I found the Hue API to be extremely well designed and a pleasure to work with. The program works great, and I'm still getting feedback from users and adding features as they are requested.

ScreenBloom is great to throw on for a movie, where vibrant scenes will extend out into your living room. When used with video games the program really shines as the lighting gradually shifts with the perspective of the character. The effect is subtle but immersive, especially with the dynamic brightness setting enabled that dim the lights as your character enters a darker environment, like a cave.

If you have some Hues, you can head over to screenbloom.com and download the application for yourself!
