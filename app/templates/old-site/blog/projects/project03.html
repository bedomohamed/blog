Technologies Used:

Python, HTML, CSS, JavaScript, Flask, Linux, Decoupage!

This project allowed me to combine my love of computer hardware with my new found interest in programming. It's not exactly as practical as a lot of other Pi projects, but it is certainly entertaining! The work turned out to have a much wider scope than I initially prepared for, forcing me to step outside my comfort zone and implement concepts I had not yet come across, like AJAX.

The code for the project is linked above along with a separate configuration website I created for the device. The configuration website has received a functionality makeover since I first completed the project, and you can change the settings on the actual device sitting on my desk with the control panel website.

Knock yourself out!
