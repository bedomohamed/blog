Content Management System

Public Demo
Python HTML Javascript
Technologies Used:

Python, Flask, HTML, CSS, Javascript, jQuery, SQLAlchemy

I've been interested in developing my own CMS for some time. Managing numerous HTML files for my posts was quickly becoming unruly. It was tedious, time consuming, and it violated basic principles of DRY. I saw this problem as the next opportunity to continue my programming education. After SteamTime I decided I wanted to get more comfortable with databases and a CMS would surely be one way to accomplish that goal.

This project was about pulling everything I had been learning together to form a cohesive piece of software that is functional and easy to use. The CMS project has furthered my comfort level with languages like Python and JavaScript, as well as given me plenty of experience with databases and the issues that come along with them.

At the end of the day I've developed a tool that saves me a lot of time and is fun to use.

I've written a few articles on some of the features of the CMS and the issues I had to tackle to complete the project which can be viewed below.

Feel free to peruse the CMS for yourself. The link above leads to a public facing replica of the same CMS I use for the blog, so please test it out and see what you think!
