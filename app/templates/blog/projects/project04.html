GIF Party!

Live Site
Python HTML CSS (Welcome Page) CSS (Application) Javascript
Technologies Used:

Python, HTML, CSS, JavaScript, jQuery, AJAX, Flask

GIF Party was a pet project to help me get comfortable with JavaScript. I have all these GIF links optimized and readily available from my previous GIF Display project and I wanted a fun way to display them. I came up with the idea for a full-screen web app where GIFs pulled from my master list will randomly fly around the screen creating a ridiculous cacophony of internet madness. The app is in a very early state as I can only devote a small chunk of my spare time to its development, but it works!

One of the main reasons I'm so invested in this app is that it's going to serve as my JavaScript trial by fire. I'm continually improving and expanding the code which runs the app.

Bandwidth warning: the maximum number of GIFs the app can display at one time is 20, which averages out to about 50 MB of data.
