GIF Party!

Live Site
Python HTML CSS (Welcome Page) CSS (Application) Javascript
Technologies Used:

Python, HTML, CSS, JavaScript, jQuery, AJAX, Flask

GIF Party was a pet project to help me get comfortable with JavaScript.

I have thousands of optimized GIF links from my previous GIF Display project and I wanted a fun way to display them. I came up with the idea for a full-screen web app where GIFs pulled from my master list will randomly fly around the screen creating an absurd bit of internet madness.

Gif Party expanded my JavaScript knowledge and got me comfortable with concepts like grid-based animation, DOM manipulation, and made me really pay attention to the user interface and experience.

Bandwidth warning: the maximum number of GIFs the app can display at one time is 20, which averages out to about 50 MB of data. Depending on the auto-refresh setting, this can add up quick, so please be aware if you've got a data limit.
