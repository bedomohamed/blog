GIF Party!

Live Site  Github


Technologies Used:

Python, HTML, CSS, JavaScript, jQuery, Flask

GIF Party is my current pet project. I have all these GIF links optimized and readily available from my previous GIF Display project and I wanted a fun way to display them. I came up with the idea for a full-screen web app where GIFs pulled from my master list will randomly fly around the screen creating a ridiculous cacophony of internet madness. The app is in a very early state as I can only devote a small chunk of my spare time to its development, but it works!

One of the main reasons I'm so invested in this app is that it's going to serve as my JavaScript trial by fire. Currently I have some copy-pasta JS controlling the GIFs' movements around the page but there are some serious problems with it. I'll need to dig deep into JavaScript and jQuery to get my bearings and start making progress with this thing. Can't wait!

Bandwidth warning: the maximum number of GIFs the app can display at one time is 20, which averages out to about 50 MB of data. Keep in mind that the JavaScript to control the GIFs' movements doesn't load until the entire page has finished loading, so if you're displaying 20 GIFs it might take a moment for the images to start moving around as they are still downloading.
