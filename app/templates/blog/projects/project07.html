SteamTime

Live Site
Python HTML CSS Javascript
Technologies Used:

Python, HTML, CSS, Javascript, jQuery, Flask

SteamTime spawned from some API doodling I was doing back in September.

I wanted a way to easily visualize data returned by the API, so I set about exploring the interfaces and quickly built a simple CLI program to list a user's most played Steam games. From there the project grew and grew and I found myself plugging in all kinds of stuff like JavaScript libraries and CSS transitions. The end result is a cohesive and full-featured app, and I'm quite proud of the finished product.

A few days after I completed the project I got some feedback from Reddit and refactored the code. I wrote a quick blog post about it and have saved both versions for posterity.
