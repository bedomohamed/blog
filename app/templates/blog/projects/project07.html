Technologies Used:

Python, HTML, CSS, Javascript, jQuery, Flask

SteamTime spawned from some API doodling I was doing back in September. I wanted a way to easily visualize data returned by the API, so I set about exploring the interfaces and quickly building a simple CLI program to list a user's most played Steam games. From there the project grew and grew and I found myself plugging in all kinds of stuff like JavaScript libraries and CSS transitions. The end result is my most cohesive and full-featured app yet, and I'm quite proud of the finished product and everything I learned in the process.
